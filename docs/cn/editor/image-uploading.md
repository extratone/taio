# 图片上传

您有很多种方法在文档中嵌入图片，它们在您的设备上工作得很完美，但它们是本地文件。

当您把文件移到另一个位置，或把内容作为博客文章发布时，图片需要上传到服务器上。

## 图床动作

[Image Uploader](https://github.com/cyanzhong/Image-Uploader) 项目就是为此而设计的，您可以把图片上传到任何您想要的服务器上，使用专门为 Markdown 写作设计的直观界面。
